Paleta bimodal · light + dark

Cada categoría tiene ahora dos variantes: una tuneada para crema (light) y otra para grafito (dark). El sistema elige automáticamente según el tema activo. Mismo patrón que ya siguen --crema-* y --ink-*.

Tecla D alterna light/dark. La etiqueta verde renderiza en este tema marca cuál variante es la que se vería con el modo actual.
