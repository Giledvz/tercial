Despejes

Nivel principiante

  1. $v = \dfrac{d}{t}$;  $d = v\,t$,  $t = \dfrac{d}{v}$
  2. $F = G\,\dfrac{m_1\,m_2}{r^2}$;  $m_1 = \dfrac{F\,r^2}{G\,m_2}$,  $r^2 = \dfrac{G\,m_1\,m_2}{F}$
  3. $F = m\,\omega^2\,r$;  $m = \dfrac{F}{\omega^2\,r}$,  $r = \dfrac{F}{m\,\omega^2}$
  4. $U = \dfrac{1}{2}\,k\,x^2$;  $k = \dfrac{2U}{x^2}$,  $x^2 = \dfrac{2U}{k}$
  5. $V = \pi\,r^2\,h$;  $r^2 = \dfrac{V}{\pi\,h}$,  $h = \dfrac{V}{\pi\,r^2}$
  6. $\dfrac{P_1V_1}{T_1} = \dfrac{P_2V_2}{T_2}$;  $P_1 = \dfrac{P_2V_2\,T_1}{V_1\,T_2}$,  $T_2 = \dfrac{P_2V_2\,T_1}{P_1V_1}$
  7. $W = F\,d\,\cos\theta$;  $F = \dfrac{W}{d\,\cos\theta}$,  $d = \dfrac{W}{F\,\cos\theta}$
  8. $A_1V_1 = A_2V_2$;  $A_2 = \dfrac{A_1V_1}{V_2}$,  $V_2 = \dfrac{A_1V_1}{A_2}$
  9. $\tau = F\,r\,\sin\theta$;  $F = \dfrac{\tau}{r\,\sin\theta}$,  $r = \dfrac{\tau}{F\,\sin\theta}$
  10. $K = \dfrac{1}{2}\,m\,v^2$;  $m = \dfrac{2K}{v^2}$,  $v^2 = \dfrac{2K}{m}$
  11. $x = \dfrac{1}{2}\,g\,t^2$;  $g = \dfrac{2x}{t^2}$,  $t^2 = \dfrac{2x}{g}$
  12. $s = \dfrac{d}{t-t_0}$;  $d = s\,(t-t_0)$,  $t-t_0 = \dfrac{d}{s}$
  13. $U = mgh$;  $m = \dfrac{U}{gh}$,  $h = \dfrac{U}{mg}$
  14. $F = k\,\dfrac{q_1q_2}{r^2}$;  $q_1 = \dfrac{F\,r^2}{k\,q_2}$,  $r^2 = \dfrac{k\,q_1q_2}{F}$
