El producto para el alumno inscrito: abre con HOY (repaso acumulativo + contexto del curso, tu no-negociable hecho pantalla) y debajo tiene todo el material por aprende/practica/simula. Cero venta — eso vive en una página aparte. Trae control para previsualizar otros días.

La ruta

Idea nueva · a partir de la propuesta C

El plan de hoy · auto-servicio

El alumno abre y sabe qué hacer HOY: el plan sale solo del calendario del curso (¿hay clase? ¿asesoría? ¿día libre?) + un ritmo semanal, anclado a aprende/practica/simula. Fijo e igual para todos; mañana cambia solo. Trae un control para previsualizar otros días.
